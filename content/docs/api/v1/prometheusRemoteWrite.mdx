Use this route when metrics are being forwarded with the Prometheus Remote Write protocol.

This endpoint accepts the standard snappy-compressed Remote Write payload. `X-P-Stream` chooses the destination dataset, and `X-P-Log-Source: otel-metrics` tells Parseable to store the converted payload as metrics data.
