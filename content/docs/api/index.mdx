# Parseable API Reference

Welcome to the Parseable API documentation. This section provides detailed information about the available API endpoints for interacting with Parseable.

## Getting Started

Before you begin, make sure you have:
- A running instance of Parseable
- The base URL of your Parseable server
- Required authentication credentials (if authentication is enabled)

## Authentication

All API requests require authentication. Include your API key in the `Authorization` header:

```bash
Authorization: Bearer YOUR_API_KEY
```

## Rate Limiting

API requests are subject to rate limiting. The default rate limit is 100 requests per minute per IP address.

## Error Handling

All error responses follow a standard format:

```json
{
  "error": {
    "code": "error_code",
    "message": "Human-readable error message"
  }
}
```
