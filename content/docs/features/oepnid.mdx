Parseable supports OpenID Connect (OIDC) authentication for secure access to all its functionality. Parseable uses OpenID connect discovery mechanism to connect to Identity providers.

## Key Features

- **SSO Integration** - Connect with any OIDC-compliant identity provider (Google, Okta, Auth0, etc.)
- **Group-based access** - Map identity provider groups to Parseable roles
- **Multi-role support** - Assign multiple roles to a single user for fine-grained access control
- **Default roles** - Set default permissions for new OIDC users

## Supported Providers

Parseable works with any OIDC-compliant identity provider including:

- Google Workspace
- Okta
- Auth0
- Azure AD
- Keycloak
- And more...
