description: Send logs to Parseable from your application code

Parseable supports direct log ingestion from various programming languages using HTTP clients or dedicated SDKs:

- [Go](./go) - Native Go HTTP client integration
- [Python](./python) - Python logging with HTTP transport
- [Java](./java) - Java logging frameworks integration
- [JavaScript](./javascript) - Node.js and browser logging
- [Rust](./rust) - Rust HTTP client integration
- [C#](./csharp) - .NET logging integration
- [.NET](./dotnet) - .NET Core and Framework support
- [PHP](./php) - PHP logging integration
- [Ruby](./ruby) - Ruby logging integration
