---
title: Logging Agents
description: Send logs to Parseable using popular logging agents
---

Parseable supports ingestion from various logging agents. Choose the agent that best fits your infrastructure:

- [Fluent Bit](./fluent-bit) - Lightweight and high-performance log processor
- [Vector](./vector) - High-performance observability data pipeline
- [Fluentd](./fluentd) - Unified logging layer
- [Logstash](./logstash) - Server-side data processing pipeline
- [Filebeat](./filebeat) - Lightweight shipper for logs
- [Promtail](./promtail) - Agent for Loki, works with Parseable
- [OpenTelemetry Collector](./otel-collector) - Vendor-agnostic telemetry collection
- [Syslog](./syslog) - Standard logging protocol
- [Apache Log4j](./apache-log-4j) - Java logging framework
- [Prometheus](./prometheus) - Metrics collection and forwarding
